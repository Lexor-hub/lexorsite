# Lexor Site

Site institucional premium da Lexor, construído com Next.js App Router, Tailwind CSS e Framer Motion.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Configurar WhatsApp

Troque o valor de `WHATSAPP_NUMBER` em `src/components/lexor-experience.tsx` pelo número oficial da Lexor em formato internacional, apenas dígitos.
